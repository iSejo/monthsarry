# Monthsary Project Sharing

This project is a static HTML page with local images and can be opened from a mobile browser if hosted as a regular web page.

## Best mobile-friendly option

1. Upload the entire `monthsary-project` folder to a static host:
   - GitHub Pages
   - Netlify (drag-and-drop deploy)
   - Vercel
   - any free static site host

2. Share the generated URL. Example:
   - `https://<your-username>.github.io/<repo>/`
   - `https://<random-name>.netlify.app`

## If you want the easiest one-click host

### Netlify Drop
1. Go to https://app.netlify.com/drop
2. Drag the `monthsary-project` folder onto the page
3. Wait for deployment to finish
4. Copy the link and send it to your girlfriend

### GitHub Pages
1. Create a GitHub repository and push the `monthsary-project` folder content
2. Enable Pages from the repository settings
3. Set the source to `main` branch and `/ (root)`
4. Use the generated URL and send it to her

## Notes
- Do not send the `.zip` file directly if she only has a phone; she may not be able to unzip and open the page.
- The mobile-friendly way is to send a browser URL so she can tap it and see the monthsary page.
